# BE-API

### Setup Env
Copy form `.env.example` to `.env` and setup databse

## Run Project
### Install Dependencies

```
yarn install
```

### Migration
```
yarn db:migrate
```

### Development

```
yarn start:dev
```

### Production
```
yarn build 
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.
